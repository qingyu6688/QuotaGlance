# 社区行为准则

QuotaGlance 希望维护一个友好、专业、可协作的开源社区。

## 我们鼓励

- 围绕事实、代码和用户场景讨论问题。
- 尊重不同经验、背景、语言和技术选择。
- 给出可复现步骤、建设性建议和清晰的审查意见。
- 主动承认错误并帮助修复社区中的误解。

## 不可接受的行为

- 人身攻击、歧视、骚扰、威胁或持续干扰讨论。
- 未经允许公开他人的私人信息。
- 发布凭据、Token、Cookie、账号响应或其他敏感数据。
- 垃圾信息、恶意代码、供应链投毒或绕过项目安全边界。

## 执行

维护者可以编辑、隐藏或删除违反准则的内容，并根据严重程度限制参与。需要私下报告的问题请发送至 maorongkang@gmail.com；报告会在合理范围内保密处理。

本准则适用于仓库、Issue、Pull Request、Release 讨论及代表项目参与的其他公开空间。
